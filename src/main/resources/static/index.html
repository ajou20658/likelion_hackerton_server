How to use hackerton api

crawling api

  • /api/crawl
    • 어제 하루동안의 기사 크롤링
  • /api/save-news?sid1=
    • 100번부터 105번까지 기사 mongodb에 저장

Flask server

  • /api/keyword
    • 저장된 기사 txt파일에 접근하여 키워드 추출후 mongodb에 저장

etc api

  • /api/save-summary?sid1=
    • mongodb에 저장된 기사의 본문을 크롤링후 naver api사용하여 요약후 mongodb에 저장
  • /api/summary?url=
    • url을 넘겨주면 본문을 크롤링하고 naver api를 사용하여 요약후 반환
  • /api/search?keyword=
    • keyword를 넘겨주면 해당 키워드에 관련된 기사들을 크롤링후, 본문을 요약하여 db에 저장 s
    이후 같은 키워드를 검색하면 이전에 저장한 기사를 출력함
